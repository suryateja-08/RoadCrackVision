#  RoadCrackVision
### Deep Learning Based Road Crack & Pothole Detection System

RoadCrackVision is a deep learning-powered web application that detects road cracks and potholes from images using Convolutional Neural Networks (CNN) and transfer learning models. This system can assist smart city initiatives and road maintenance authorities in automating infrastructure inspection.

---

##  Features

- Upload road images via web interface
- Detect cracks and potholes using trained deep learning models
- Multiple model support (CNN, MobileNet, InceptionResNet)
- Real-time prediction using Flask backend
- Clean and simple UI
- Lightweight GitHub repository (models hosted externally)

---

##  Models Used

This project includes the following trained models:

- Custom CNN Model
- MobileNet (Transfer Learning)
- InceptionResNet (Transfer Learning)

>  Due to GitHub’s 100MB file size limitation, trained model files are hosted externally.
